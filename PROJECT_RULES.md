# ATLAS - Project Rules

## Vision

Atlas est l'application de référence pour vivre, voyager et s'installer au Maroc.

Notre objectif est de construire le meilleur compagnon numérique pour les MRE, les touristes et les expatriés.

## Priorités

Toujours privilégier :

- la simplicité
- la rapidité
- la lisibilité
- les performances
- le design premium

## Architecture

Le code doit toujours être :

- modulaire
- facilement testable
- documenté
- évolutif

## UX

Chaque écran doit répondre à une vraie question que se pose l'utilisateur.

## Style

Utiliser Material 3.

Éviter les widgets complexes si une solution simple existe.

Toujours commenter les parties importantes.

## Philosophie

Nous ne développons pas une application.

Nous développons un compagnon de vie.

Chaque nouvelle fonctionnalité doit répondre à cette question :

"Est-ce que cette fonctionnalité réduit le stress ou fait gagner du temps à l'utilisateur ?"

Si la réponse est non, elle ne fait pas partie du MVP.
